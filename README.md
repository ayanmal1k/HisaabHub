HisaabHub is a high-performance, visually stunning financial ecosystem built with **React Native (Expo)**. Designed for professionals and individuals who demand precision, it integrates persistent local ledger tracking with advanced mathematical tools into a cohesive "Glassmorphism" interactive experience.

---

## 🎨 Design Philosophy
HisaabHub is built on the principle of **Tactile Excellence**. Every button press uses spring-physics animations to provide instant, meaningful feedback. The interface utilizes a curated palette of bluish-cyan accents, soft shadows, and semi-transparent layers to create a premium, state-of-the-art workspace.

## 🚀 Versatile Toolset

### 🔢 1. Advanced Calculator
A powerhouse for daily math, featuring custom-built logic that handles percentages and complex expressions in real-time.
- **Buttery-Smooth Keys**: Dedicated `Pressable` components with spring animations.
- **Live Result**: Instant evaluation of mathematical expressions as you type.
- **Smart Formatting**: Handles thousands-separators and precision rounding automatically.

### 💰 2. Fiverr Professional Suite
A specialized tool for freelancers to calculate exactly what they take home after fees.
- **Conditional Fee Logic**: Toggle between **Payoneer** and **Direct** modes.
- **Breakdown Audit**: Transparently shows Fiverr's 20% cut, fixed $3 withdrawal fees, and 2.5% processing charges.
- **Live Conversion**: Converts net USD earnings to PKR based on a high-precision exchange rate.

### 📈 3. Earning Tracker (Monthly Ledger)
A persistent income tracker that helps you manage monthly payouts and pending receipts.
- **History Logs**: Every adjustment (+/-) is recorded with a timestamp and custom label.
- **Visual Insights**: A top-level summary bar shows your **Total Lifetime Earnings** and tracking range.
- **Smart Validation**: Prevents duplicate month entries to ensure data integrity.

### 🤝 4. Udhaar (Debt & Credit) Tracker
A simplified, professional way to manage personal loans and credits.
- **Net Balance Cards**: Instantly see "You Get" vs. "You Give" with color-coded summary boxes.
- **Interactive Transactions**: Add lent or borrowed amounts with one-tap adjustments.
- **Contact Management**: Minimalist, text-focused contact list for rapid navigation.

## 🛠 Technical Foundation
- **Core Technology**: React Native (Expo) - for high-performance cross-platform interactions.
- **Data Privacy**: **Offline-first** architecture. All data stays securely on your device via `AsyncStorage`.
- **Navigation**: Custom gesture-based tab system with circular active indicators.
- **Graphics**: Lucide-React-Native for crisp, consistent iconography.

## 📦 Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run the Suite**:
   ```bash
   npx expo start
   ```

3. **Explore**:
   Swipe between the four specialized tabs and start your hisaab today!
